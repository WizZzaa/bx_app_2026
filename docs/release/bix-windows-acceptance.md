# Приёмка виджета Бикса — Windows 10/11

Проверять на чистой установке `BX-win32-x64-Setup.exe`, под обычной учётной записью BX.

## Окно и панель задач

- [ ] После запуска Бикс виден над нижней панелью задач; видны голова и свисающие лапки.
- [ ] Перетаскивание за голову работает и позиция сохраняется после перезапуска.
- [ ] Ручное растягивание за каждый край и угол работает; размер сохраняется после перезапуска, а лапки остаются у панели задач.
- [ ] На масштабе 100 %, 125 %, 150 %, 175 % и 200 % персонаж не обрезается и не перекрывает системные кнопки.
- [ ] На втором мониторе виджет не пропадает и остаётся возле панели задач выбранного экрана.
- [ ] При панели задач снизу, сверху, слева, справа и в режиме автоскрытия окно остаётся в рабочей области выбранного монитора.
- [ ] Виджет остаётся поверх обычных окон BX, но не мешает нажимать элементы панели задач.

## Основные сценарии

- [ ] Нажатие на Бикса открывает веер: AI, задача, заметка, утилиты, переводчик и домик.
- [ ] Быстрая задача появляется в Планировщике; заметка сохраняется после перезапуска.
- [ ] Каждая из шести утилит открывает соответствующий раздел BX.
- [ ] Напоминание о просроченной/сегодняшней задаче даёт действия «Готово», «15 мин», «Открыть».
- [ ] «Готово» меняет статус задачи, «15 мин» переносит напоминание, без дублей.
- [ ] При открытом Биксе напоминание появляется только в его облачке; после закрытия облачка не возникает запоздалый Windows-toast.

## Бикс и приватность

- [ ] Шутки не появляются чаще выбранного интервала и исчезают примерно через 7 секунд.
- [ ] Отключение шуток, тихие часы, приватный текст напоминаний и уменьшение анимации сохраняются после перезапуска.
- [ ] В тихие часы Бикс не показывает шутки; системный режим Windows «Не беспокоить» пока намеренно не учитывается.
- [ ] Ежедневные монеты: Free 0, Standard 5, Premium 15; повторное открытие в тот же день не начисляет повторно.
- [ ] Корм и игрушка уменьшают монеты только при достаточном балансе и обновляют показатели питомца.

## Гардероб и сеть

- [ ] После входа загружается гардероб; покупка списывает монеты один раз и вещь остаётся после перезапуска.
- [ ] Гардероб разделяет образы и аксессуары; десять шляп и три дополнительных образа видны на Биксе, надетую вещь можно снять.
- [ ] Сундук показывает только купленные вещи; закрытые по тарифу предметы нельзя купить.
- [ ] При отсутствии сети быстрые заметки не теряются, а сетевые действия показывают понятное сообщение без падения виджета.
- [ ] Кормление, игра и ежедневный подарок, сделанные без сети, попадают в очередь и после подключения применяются ровно один раз.
- [ ] После повторного входа и на втором компьютере баланс, потребности и инвентарь восстанавливаются из серверного состояния.

## Достижения

- [ ] В Домике открывается список из 9 достижений с прогрессом, условием и наградой.
- [ ] Создание/завершение задач, вопросы AI, коллекция, забота и ежедневные подарки увеличивают только соответствующий серверный прогресс.
- [ ] Достигнутая цель начисляет награду один раз; повторная синхронизация, перезапуск и второй компьютер не дублируют монеты.

## Что приложить к отчёту

Для каждого дефекта: версию BX, Windows 10/11, масштаб экрана, один/два монитора, шаги, ожидаемый и фактический результат, скриншот или короткую запись.
